Headings

Headings define the outline of your site. There are six levels of headings.

Unordered List

  • Add "ul" element (it stands for unordered list)
  • Add each item and its own "li" element
  • They don't need to be in particular order

Ordered List

  1. Notice "ol" element wrapping everything
  2. But, the list item elements are the same
  3. Also note how the numbers increments on their own
  4. You should be noticing things is this precise order, because this is ordered list
